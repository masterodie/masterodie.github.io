Mod Organizer Installation

Vorwort

Mod Organizer ist ein Tool zum verwalten von Mods für Spiele wie Fallout 3, Fallout New Vegas, The Elder Scrolls IV - Oblivion und The Elder Scrolls V - Skyrim. Es hat einige Vorteile gegenüber dem sonst üblicherweise verwendeten Nexus Mod Manager, ist allerdings auch etwas komplizierter zu benutze. Allerdings lohnt sich die etwas längere Einarbeitungszeit die man für Mod Organizer braucht wenn man z.B. mehrere Profile für verschiedene Mod Konfigurationen haben will (z.B. ein Profil zum normalen Spielen und eins mit vielen Grafik Mods für Screenshots, oder für verschiedene Overhaul Mods die man gleichzeitig spielen will). Diese Möglichkeit bietet Nexus Mod Manager zwar auch seit kurzem, allerdings ist dies dort nicht ganz so praktisch gelöst wir bei Mod Organizer.

Dies ist der Fall durch die Art und Weise in der Mod Organizer Mods intern verwaltet. Mod Organizer legt für jeden Mod den man installiert in einem von Mod Organizer verwalteten Verzeichnis ein eigenes Unterverzeichnis an, dessen Inhalt erst beim starten eines Programmes über das Mod Organizer Interface in das Data Verzeichnis des jeweiligen Spieles zur Verfügung stellt. Das bedeutet man muss beim installieren von Mods nie mehr drauf achten welcher Mod welche Dateien von anderen Mods überschreibt, dies wird intern von Mod Organizer so gelöst das es etwas ähnliches wie die Load Order von Fallout 3, Fallout New Vegas, The Elder Scrolls IV - Oblivion oder The Elder Scrolls V - Skyrim. Das heißt man kann die installierten Mods im Interface beliebig hin und her sortieren wie man das auch mit den von Mods installierten *.ESP Dateien kennt, Mods die eine höhere Priorität in dieser zweiten Load Order haben werden dann bevorzugt behandelt und Dateien dieser Mods werden in der Standard Einstellung im Data Verzeichnis vom Spiel landen und genutzt werden. Dieses Verhalten kann man allerdings auch pro Mod pro Datei noch mal einstellen, so das man jegliche Freiheiten hat welche Dateien denn letztendlich vom Spiel geladen werden. Das alles macht Mod Organizer unterm Strich doch deutlich mächtiger als Nexus Mod Manager werden lässt.

Das bedeutet auch das, dass Data Verzeichnis mit Mod Organizer nicht verändert wird, und man dieses auch im Urzustand belasse kann, auch wenn man z.B. die Master Dateien von DLC's säubert oder ähnliches. Somit kann man trotz allem immer noch mit Steam die lokalen Dateien auf Fehler überprüfen, und muss nicht jedes mal danach wieder alle Änderungen an den Master Dateien durchführen muss.

Installation

Für Mod Organizer gibt es 2 mögliche Arten der Installation. Einmal kann Mod Organizer global im System installiert werden und für jedes installierte Spiel wird im Benutzer Verzeichnis ein Unterverzeichnis angelegt in dem Mods und Einstellungen für das jeweilige Spiel gespeichert werden. Alternativ kann man für jedes Spiel das man installiert hat eine separate Installation von Mod Organizer anlegen. Da werden die Mods in einem Unterverzeichnis von Mod Organizer abgelegt. Letzteres ist die von mir favorisierte Methode.

Globale Installation

Mod Organizer Installer

Dazu lädt man sich einfach von Nexus Mods die aktuelle Installer Version von Mod Organizer runter und installiert diese an einer beliebigen stelle. Beim starten wird man gefragt welche Art der Installation man denn haben möchte. Dort wählt man einfach global.

Installation pro Spiel

Mod Organizer Regulär

Dazu lädt man sich von Nexus Mods die Version von Mod Organizer runter die nicht Installer im Namen hat und entpackt diese z.B. in ein Unterverzeichnis von jeweiligen Spiel. (z.B. [Skyrim Verzeichnis]\ModOrganizer ). Beim ersten Start sollte eigentlich in diesem Fall keine Frage kommen welchen Installation Typ man denn gerne hätte. Falls diese doch kommt wählt man da einfach die manuelle Methode aus und danach ist man schon fast soweit das man Mods installieren kann.

Ersteinrichtung

Nach dem ersten Start von Mod Organizer sollte man in den Optionen, die man über das Zahnrad Symbol erreichen kann, seine Login Daten für Nexus Mods eintragen, damit man Mod Organizer dafür benutzen kann Mods herunter zu laden.

Ggf. kann man zu diesem Zeitpunkt auch noch einstellen wohin Mods herunter geladen werden sollen oder wo das Mods Verzeichnis von Mod Organizer abgelegt werden soll. Wenn man dies später ändern will muss man die ursprünglichen Verzeichnis an die neue Stelle verschieben oder alles neu installieren.

Das Benuzer Interface

Nachdem man die Einrichtung beendet hat sollte man sich erst einmal das Benutzerinterface von Mod Organizer genauer anschauen. Die Leiste oben am Rand beinhaltet auf der linken Seite Knöpfe um Mods zu importieren, oder um Einstellungen an Mod Organizer selber vorzunehmen. Auf der rechten Seite findet man eine Update Benachrichtigung und was viel wichtiger ist eine Fehleranzeige. Wenn diese nicht ausgegraut ist und eine Zahl daneben steht ist möglicherweise irgendwas im argen und man sollte sich diese Fehlermeldung auf einen klick auf das Symbol mit dem Ausrufezeichen anzeigen lassen.

Weiter ist unten links das grosse Fenster die Liste mit allen installierten Mods im Mods Verzeichnis von Mod Organizer Dies können von Mod Organizer verwaltete Dateien sein oder nicht von Mod Organizer verwaltete Dateien sein. Dateien die Skyrim selber mitbringt fallen zum Beispiel unter die letzte Kategorie, und alles was man so mit Mod Organizer herunter lädt und verwaltet und die erste. Beim ersten Start von Mod Organizer befinden sich dort nur die DLC's falls vorhanden. Die Einträge hier kann man sortieren und alles was tiefer in der Liste steht solange nach Priorität sortiert ist wird von Skyrim bevorzugt vor Einträgen die höher in der Liste sind Behandelt. Also Dateien von Mods mit höherer Priorität landen letztendlich im Data Verzeichnis von Skyrim wenn man ein Programm über Mod Organizer startet.

Rechts neben dieser Liste befindet sich oben die Dropdown Liste der Programme die Mod Organizer kennt, und daneben der Start Knopf, der das jeweils ausgewählt Programm startet. In diesem Menü befindet sich auch ein Menü Punkt mit dem man manuell neue Programme hinzufügen kann die über Mod Organizer gestartet werden können. Dazu zählen z.B. TES5Edit oder SkyProc Patcher.

Unter dem App Launcher befindet sich eine Liste mit der Load Order für die *.ESP Dateien von Skyrim. Diese kann man auch manuell sortieren oder aber über den Knopf über der Liste mit einer eingebauten Version von LOOT sortieren lassen kann. Hier kann man auch über verschiedene Reiter auf andere Funktionen zugreifen, von denen die wichtigste wohl der Download Reiter sein dürfte. Dort landen nachdem man über Nexus Mods einen Mod herunter geladen hat die Dateien und man kann darüber auch den Mod installieren.

Schlusswort
