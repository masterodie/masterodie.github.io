Installation von SKSE

Vorwort

In diesem Artikel werde ich erklären wie man SKSE so installiert das es soweit möglich von Mod Organizer verwaltet wird. Das ganze ist nicht allzu kompliziert und sollte einem die Möglichkeiten von Mod Organizer abseits von Nexus Mods etwas näher bringen.

Ähnliche Script Extender gibt es auch für Fallout 3 (FOSE), Fallout New Vegas (NVSE) und The Elder Scrolls IV - Oblivion (OBSE)

Installation

Als erstes benötigen wir die neueste Version von SKSE, dem Skyrim Script Extender. Nachdem man diese heruntergeladen hat entpackt man das Archiv an beliebige Stelle im Dateisystem.

Hier benennen wir das Data Verzeichnis um und verschieben dieses in das Mod Organizer Mods Verzeichnis, so das SKSE neben Update_CLEANED liegt. Damit sind schon mal alles Dateien im Data Verzeichnis für SKSE zu finden wenn wir es darüber starten wollen.

Als nächstes werden wir den restlichen inhalt des eben entpackten Verzeichnisses (ggf. bis auf die README Dateien und das src Verzeichnis) in das Skyrim Verzeichnis verschieben, so wie man das sonst auch machen würde wenn man SKSE Installiert.

Mod Organizer

Als nächstes startet man kurz mal Mod Organizer und man die Dateien aus dem letzten Schritt an die richtige Stelle verschoben hat sollte man oben in dem Dropdown Menü in dem man die zu startende Apllikation auswählen kann scho SKSE ausgewählt sein. Wenn nun noch das SKSE Verzeichnis in der Liste der installierten Mods auftauch muss nur noch sicher gestellt werden das dieser Mod auch aktiviert ist, was man einfach mit dem Haken links neben der Zeile anstellen kann.
